## Project Overview

- This project is an interactive web-based story titled “The Signal in the Subway.” The story takes place in the New York City subway system and follows a character who begins hearing a mysterious signal through their headphones while waiting on a late-night platform.

As the user navigates through the webpage, they make choices that affect how the story unfolds. Each decision leads to a different path, allowing the user to explore different locations such as subway platforms, train cars, hidden tunnels, and underground control rooms. Depending on the choices the user makes, the story can end in several different ways.

The goal of the project was to combine visual storytelling, sound design, and interactive web elements to create an engaging narrative experience. By clicking choices and exploring the page, users gradually uncover the mystery behind the signal.

## Features

- Button pulse animation to guide users toward interactive choices
- Fade transitions between story scenes to create a smooth narrative flow

## Sound Effects

Sound plays an important role in creating tension and immersion.

Sound effects are used for:
	•	Signal interference/static when the story begins
	•	Radio frequency sounds when navigating deeper into the subway network
	•	Ending sounds that reinforce whether the player reached a successful or failed outcome
